# 复合井字棋（Ultimate Tic-Tac-Toe）

基于 Vue 3 + WebSocket 的嵌套井字棋在线对战游戏，支持：

- 两位玩家在线对战（房间制）
- 观战者实时观看
- AI 对战模式
- 移动端支持

---

## 快速开始

### 开发模式

1. **安装依赖**：
   ```bash
   npm install
   ```

2. **启动开发服务器**：
   ```bash
   npm run dev
   ```
   访问 `http://localhost:5173`

3. **启动后端服务器**（另一个终端）：
   ```bash
   node server/index.cjs
   ```

### 生产部署

1. **构建前端**：
   ```bash
   npm run build
   ```

2. **启动服务器**：
   ```bash
   npm start
   ```
   服务器会自动提供静态文件服务（端口 3000）

服务器会同时处理：
- 静态文件服务（HTML/CSS/JS）
- WebSocket 连接（游戏通信）

---

## 本地联机测试

如果需要在局域网内其他设备访问：

1. 确保服务器已启动（`npm start` 或 `node server/index.cjs`）
2. 查看本机局域网 IP（如 `192.168.1.10`）
3. 在其他设备浏览器访问 `http://192.168.1.10:3000`

> 注意：确保防火墙允许端口 3000 访问

---

## 功能特性

- **昵称系统**：玩家可设置昵称显示在游戏中
- **观战模式**：支持多人观战并显示观战人数
- **重置游戏**：玩家可发起重置操作
- **AI 对战**：支持与 AI 对战
- **响应式设计**：支持移动端访问
- **二维码分享**：便于移动设备快速加入

---

## 技术栈

- **前端**：Vue 3 + Vite
- **后端**：Node.js + WebSocket (ws)
- **运行要求**：Node.js >= 20.19.0
